# Mini Text-Based Game Engine in Assembly

A custom text-based game engine written in x86 assembly.  
This project demonstrates low-level memory manipulation, control flow, and instruction handling to simulate a basic game environment.

---

## Features
- Player movement using registers
- Enemy encounters using pseudo-random logic
- Text output for game state
- Fetch-decode-execute loop to simulate CPU cycles

---

## Example Gameplay
Player at (2,3) Move: UP Player at (2,4) Enemy appears! Fight or run?




---

## Built With
- x86 Assembly (NASM)
- Linux syscalls
